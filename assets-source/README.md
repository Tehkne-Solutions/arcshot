# Assets Source

Coloque aqui os arquivos de produção antes do processamento.

Estrutura recomendada:

```text
characters/<id>/
weapons/<id>/
effects/<id>/
environments/<id>/
audio/<id>/
```

Enquanto uma arte definitiva não existe, a Asset Factory gera um SVG provisório a partir do manifesto correspondente. Assim, gameplay e conteúdo não ficam bloqueados por arte.
